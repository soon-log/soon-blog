# Task ID: 9
# Title: RSS 피드 생성
# Status: pending
# Dependencies: 1, 2
# Priority: low
# Description: 전체 블로그와 개별 카테고리에 대한 RSS 피드를 생성하는 시스템을 만듭니다.
# Details:
다음 기능을 가진 RSS 피드 생성 시스템을 구현합니다:
1. 모든 블로그 포스트에 대한 RSS 피드 생성
2. 각 카테고리에 대한 별도 RSS 피드 생성
3. 빌드 프로세스 중 피드 업데이트

구현 세부사항:
- scripts/generate-rss.js 생성:
  - feed 패키지를 사용하여 RSS XML 생성
  - /rss.xml에 메인 피드 생성
  - /category/[category]/rss.xml에 카테고리 피드 생성
- 피드에 포함할 내용:
  - 포스트 제목, 날짜, 작성자
  - 포스트 요약 또는 발췌문
  - 전체 포스트 링크
  - 피드 카테고리로서의 카테고리 및 태그
- 빌드 프로세스와 통합:
  - package.json scripts에 추가
  - build-posts.js 후 실행
- pages/api/rss.ts 생성 (선택사항):
  - 필요한 경우 RSS 피드를 동적으로 제공

RSS 피드 구조:
- 표준 RSS 2.0 형식 사용
- 적절한 채널 메타데이터 포함
- RFC 822에 따른 날짜 형식
- HTML 콘텐츠용 CDATA 섹션 포함

# Test Strategy:
1. 단위 테스트:
   - RSS 생성 함수 테스트
   - RSS 스키마에 대한 XML 출력 검증
   - 피드 항목 형식 테스트

2. 통합 테스트:
   - 샘플 포스트 데이터로 피드 생성 테스트
   - 카테고리 피드가 올바른 포스트를 포함하는지 확인
   - 빌드 프로세스와 통합 테스트

3. 검증 테스트:
   - W3C Feed Validator로 생성된 피드 검증
   - 인기 있는 RSS 리더에서 피드 테스트
   - 모든 필수 요소가 존재하는지 확인
