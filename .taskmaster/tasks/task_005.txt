# Details:
다음 기능을 가진 포괄적인 카테고리 네비게이션 및 필터링 시스템을 생성합니다:
1. 사이드바 또는 헤더 네비게이션에서 카테고리 표시
2. 해당 카테고리의 모든 포스트가 있는 전용 카테고리 페이지 구현
3. 카테고리 내에서 태그 기반 필터링 제공
4. 카테고리 통계(포스트 수, 최신 포스트) 표시

구현 세부사항:
- components/CategoryNav.tsx 생성:
  - 사이드바 또는 헤더 네비게이션 컴포넌트
  - 포스트 수와 함께 모든 카테고리 표시
  - 현재 카테고리 강조
- pages/category/[category].tsx 생성:
  - 카테고리 페이지용 동적 라우팅
  - 선택된 카테고리의 모든 포스트 표시
  - 태그 필터링 UI 구현
  - 카테고리 설명 및 통계 표시
- 필터링 로직 구현:
  - 태그별 클라이언트 사이드 필터링
  - 카테고리 및 태그 필터 결합
  - 공유 가능한 필터링된 뷰를 위한 URL 쿼리 매개변수
- CategoryHeader 컴포넌트 생성:
  - 카테고리 이름, 설명, 통계 표시
  - 필터링을 위한 사용 가능한 태그 표시

URL 구조:
- /category/react - 모든 React 포스트
- /category/react?tags=hooks - 'hooks' 태그가 있는 React 포스트
