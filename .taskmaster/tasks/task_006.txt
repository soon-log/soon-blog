# Details:
다음 기능을 가진 검색 시스템을 생성합니다:
1. 사용자가 제목, 태그 또는 콘텐츠로 포스트를 검색할 수 있도록 허용
2. 사용자가 입력할 때 실시간 검색 결과 제공
3. 검색 결과에서 일치하는 용어 강조
4. 고급 필터링 옵션 지원

구현 세부사항:
- components/SearchBar.tsx 생성:
  - 검색 아이콘이 있는 입력 필드
  - 검색 결과용 드롭다운
  - 키보드 네비게이션 지원
- 전용 검색 페이지용 pages/search.tsx 생성:
  - 전체 페이지 검색 결과
  - 고급 필터링 옵션
  - 검색 결과 페이지네이션
- lib/search.ts에서 검색 로직 구현:
  - 미리 생성된 데이터를 사용한 클라이언트 사이드 검색
  - 더 나은 사용자 경험을 위한 퍼지 검색
  - 제목, 태그, 카테고리, 콘텐츠 전반에 걸친 검색
  - 관련성에 따른 결과 순위 매김
- SearchResult 컴포넌트 생성:
  - 포스트 제목, 날짜, 카테고리, 일치하는 발췌문 표시
  - 일치하는 용어 강조
  - 포스트 상세 페이지 링크

최적화:
- 과도한 처리를 방지하기 위해 입력 디바운스
- Fuse.js와 같은 가벼운 검색 라이브러리 사용 고려
