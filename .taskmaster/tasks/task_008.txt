# Task ID: 8
# Title: 다크 모드 지원
# Status: pending
# Dependencies: 3, 4, 7
# Priority: low
# Description: 사용자 기본 설정에 따라 다크 및 라이트 모드를 지원하는 테마 전환 시스템을 구현합니다.
# Details:
다음 기능을 가진 테마 전환 시스템을 생성합니다:
1. 다크 및 라이트 모드 지원
2. 사용자 시스템 기본 설정 존중
3. 사용자 선택 지속
4. 모든 컴포넌트에 테마 일관성 있게 적용

구현 세부사항:
- Tailwind CSS 다크 모드 사용:
  - 다크 모드 지원을 위해 tailwind.config.js 설정
  - 테마 색상을 위한 CSS 변수 생성
- ThemeProvider 컨텍스트 생성:
  - 테마 상태(다크/라이트) 저장
  - 테마 변경 메서드 제공
  - 시스템 기본 설정 변경 감지
- 테마 지속성 구현:
  - localStorage에 사용자 기본 설정 저장
  - 초기 로드 시 저장된 기본 설정 적용
- ThemeToggle 컴포넌트 생성:
  - 태양/달 아이콘이 있는 버튼
  - 상태 간 애니메이션 전환
- 모든 컴포넌트에 테마 스타일 적용:
  - 마크다운 콘텐츠
  - 코드 블록
  - UI 요소

접근성 고려사항:
- 스크린 리더에게 테마 변경 알림
- 두 테마 모두에서 충분한 대비 유지

# Test Strategy:
1. 컴포넌트 테스트:
   - ThemeToggle 기능 테스트
   - ThemeProvider 컨텍스트 동작 테스트

2. 통합 테스트:
   - 페이지 네비게이션 간 테마 지속성 테스트
   - 시스템 기본 설정 감지 테스트
   - 모든 컴포넌트에 테마 적용 테스트

3. 시각적 테스트:
   - 모든 컴포넌트가 두 테마에서 올바르게 렌더링되는지 확인
   - 전환 애니메이션 테스트
   - 대비 비율이 접근성 표준을 충족하는지 확인
