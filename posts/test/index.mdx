# 함께 일하고 싶은 동료

주변 사람들이 일에 몰입할 수 있도록 좋은 환경으로 지속적으로 개선하는 사람이다.

예를들어:

- 팀에서 비효율적으로 작업하는 건이 있다면 개인 시간을 내서 자동화를 진행하기
- PM, 디자인, 개발간 논의 내용이 정리가 안된다면 이를 정리해주기
- 개발자가 아닌 다른 사람이 봐도 이해할만큼 상세하게 문서화하기
  - PR 메시지, 개발 문서, 슬랙 메시지 등
